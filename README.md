# MSR式箱庭ゲーム Ver.4.6

## 概要

本作品は、島の知事として地形を整え、施設を建設し、人口を増やしながら島を発展させていく経営シミュレーションゲームです。
他島との交流や、突如現れる怪獣との戦い、数々の災害を乗り越え、自分だけの理想の島を作り上げましょう。

## 基本システム

* **ターン制**: 「ターンを進める」ボタンを押すことで、計画キューに入れた行動が実行され、島が1ターン進行します。
* **リソース**:
  * **資金 (G)**: 施設の建設や軍艦の建造に必要です。工場などから得られます。
  * **食料**: 人口を維持・増加させるために必要です。農場から得られます。
  * **人口**: 島の規模を表します。住宅を建てることで増加します。
  * **実績Pt**: ミッションをクリアすることで獲得できます。



## 操作方法

1. **タイルの選択**: マップ上のタイルをクリックして選択します。
2. **計画の選択**: 「計画一覧」から実行したいコマンド（農場建設、整地など）を選びます。
3. **決定**: 「決定」ボタンを押すと、計画キューに登録されます（最大20件）。
4. **ターン進行**: 「▶ ターンを進める」ボタンでキューの先頭から計画が実行されます。
5. **計画の撤回**: 計画キュー内の項目をクリックすることで、実行前の計画を取り消せます。

## 地形と施設

### 地形

* **海**: 埋め立てて陸地にしたり、掘削して油田を探したりできます。
* **平地 / 荒地**: 施設を建設するための土台です。
* **森**: 伐採して平地にできます。砲台などは他島から見ると森に偽装されます。
* **山**: 災害「隆起」などで発生します。

### 主要施設

* **農場 (100G)**: 食料を生産します。
* **工場 (100G)**: 資金を生産します。
* **港 (3000G)**: 軍艦の母港となります。
* **砲台 (1200G)**: 他島や怪獣への攻撃に必要です。
* **防衛施設 (5000G)**: 敵の攻撃から島を守ります。

## 軍艦システム

港がある場合、「軍艦建造」が可能です。

* **カスタマイズ**: 耐久度、主砲、魚雷、対空砲、弾薬庫などを予算に応じて自由に設計できます。
* **運用**: 燃料や弾薬の補給、損傷した際の修理が必要です。
* **派遣**: 「観光者コード」を使用して、他島へ軍艦を派遣させることができます。
* **状態異常**: ダメージを受けると「火災」「浸水」などの異常が発生することがあります。

## 災害と怪獣

島が発展すると、さまざまな試練が訪れます。

* **自然災害**: 台風、隕石、噴火、地震・津波、経済危機など。
* **怪獣**: 人口が増えると「シマオロシ」や「ヴォルカガロス」などの怪獣が出現します。放置すると島が荒らされるため、砲撃や軍艦で討伐する必要があります。

## セーブ・ロードについて

本ゲームはブラウザのメモリ上にデータを保持します。
* **セーブ**: 「セーブ」ボタンを押すとテキストエリアに英数字のコード（セーブデータ）が出力されます。これをコピーしてメモ帳などに保存してください。
* **ロード**: 保存しておいたコードをテキストエリアに貼り付け、「ロード」ボタンを押すと再開できます。
